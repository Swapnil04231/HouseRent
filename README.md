# 🏠 HouseRent – MERN Stack Rental Management System

## 📌 Project Overview
HouseRent is a full-stack MERN (MongoDB, Express.js, React.js, Node.js) web application designed to simplify house rental and property listing management.

It allows users to browse properties, book rentals, and allows admins to manage listings with role-based authentication and secure access.

---

## 🚀 Features

### 👤 User
- Register & Login (JWT Authentication)
- Browse properties
- Filter by location, price & type
- Book properties
- Dashboard management

### 🛡️ Admin
- Approve/Reject properties
- Manage users
- Monitor bookings

---
